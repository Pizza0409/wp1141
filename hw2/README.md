# 國際象棋遊戲

一個使用 React TypeScript 開發的現代化國際象棋遊戲，具有 AI 對手、提示功能和美觀的淺色系界面。

## 功能特色

- ✅ **完整的象棋規則**：支持所有棋子的標準移動規則
- ✅ **AI 對手**：智能 AI 對手，具有評分系統和策略思考
- ✅ **雙人模式**：支持兩個人輪流下棋
- ✅ **遊戲提示**：為玩家提供移動建議
- ✅ **重新開始**：隨時重新開始遊戲
- ✅ **新遊戲**：快速開始新的一局
- ✅ **將軍檢測**：自動檢測將軍、將死和和棋
- ✅ **被吃棋子顯示**：顯示雙方被吃的棋子
- ✅ **響應式設計**：適配各種屏幕尺寸
- ✅ **現代化 UI**：淺色系設計，簡潔美觀

## 技術棧

- **React 18** - 前端框架
- **TypeScript** - 類型安全
- **Vite** - 構建工具
- **CSS3** - 樣式設計

## 遊戲說明

### 基本操作

1. **選擇棋子**：點擊要移動的棋子
2. **移動棋子**：點擊目標位置
3. **取消選擇**：再次點擊已選中的棋子

### 遊戲模式

- **AI 模式**：與 AI 對手對戰（默認）
- **雙人模式**：兩個人輪流下棋

### 控制按鈕

- **新遊戲**：開始一局全新的遊戲
- **重新開始**：重置當前遊戲
- **提示**：獲得移動建議
- **AI 模式/雙人模式**：切換遊戲模式

### 遊戲狀態

- **輪到白方/黑方**：顯示當前輪到哪一方下棋
- **將軍**：當國王被攻擊時顯示
- **將死**：當無法解除將軍時顯示
- **和棋**：當無法移動且未被將軍時顯示

│   └── ai.ts            # AI 邏輯
├── App.tsx              # 主應用組件
├── App.css              # 應用樣式
├── main.tsx             # 應用入口
└── index.css            # 全局樣式

## AI 對手說明

AI 對手使用簡單的評分系統：

- **吃子優先**：優先選擇能吃子的移動
- **中心控制**：偏好控制棋盤中心
- **安全考慮**：避免將自己的棋子置於危險位置
- **將軍獎勵**：給予將軍額外分數
- **隨機性**：添加隨機因素避免過於可預測

## 開發說明

### 添加新功能

1. 在 `types/chess.ts` 中定義相關類型
2. 在 `utils/chessLogic.ts` 中實現邏輯
3. 在 `hooks/useChessGame.ts` 中管理狀態
4. 在組件中使用新功能

### 自定義樣式

主要樣式在 `src/App.css` 中，使用 CSS 變量可以輕鬆調整顏色主題。
