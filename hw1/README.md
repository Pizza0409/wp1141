# 個人網站


## 設計理念

本網站採用求職導向的設計理念，目標是讓訪客（特別是招聘者）能夠快速了解我的專業能力、學經歷和個人特色。設計風格簡潔專業，使用冷色調配色方案（深藍、黑色），營造出專業且現代的視覺效果。

### 核心設計原則
- **專業性**: 突出學術背景和技術技能
- **簡潔性**: 避免過度裝飾，專注於內容呈現
- **國際化**: 支援中英文雙語切換
- **響應式**: 適配各種裝置尺寸
- **未來擴展性**: 預留副業展示空間（如滑雪教練）


## 功能特色

### 核心功能
- **雙語支援**: 中英文切換，預設中文版本
- **響應式設計**: 支援桌面、平板和手機裝置
- **現代化UI**: 簡潔美觀的冷色調設計風格
- **互動動畫**: 滾動動畫、打字效果、技能條動畫
- **智能導航**: 平滑滾動、回到頂部按鈕、頁面導航選單

### 內容展示
- **個人簡介**: 包含學術背景和運動興趣
- **學經歷時間線**: 按時間順序展示教育和工作經歷
- **技能分類**: 程式語言、技術框架與工具、軟技能
- **聯絡資訊**: 電話、Email、GitHub、LinkedIn

### 特殊設計
- **重疊時間線**: 支援同時進行的活動（如學業與工作）
- **技能動畫**: 動態顯示技能熟練度
- **外部連結**: 社交媒體連結在新標籤頁開啟

## 使用方式

1. 直接在瀏覽器中開啟 `index.html` 檔案
2. 使用右上角的語言切換按鈕在中英文間切換
3. 滾動頁面查看不同區塊內容
4. 使用右下角的導航按鈕快速跳轉到特定區塊
5. 點擊回到頂部按鈕快速返回頁面頂端


## 技術規格

### 前端技術
- **HTML5**: 語義化標記，支援雙語內容
- **CSS3**: Flexbox、Grid、動畫效果、響應式設計
- **JavaScript ES6+**: 類別、箭頭函數、DOM操作
- **無外部依賴**: 純原生技術實現

### 設計特色
- **冷色調配色**: 深藍 (#1e3a8a)、黑色 (#0f172a)、亮藍 (#3b82f6)
- **響應式布局**: 移動優先設計原則
- **動畫效果**: CSS動畫和JavaScript互動
- **無障礙設計**: 良好的對比度和可讀性

## 瀏覽器支援

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+
