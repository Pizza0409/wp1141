# 國際象棋遊戲

一個使用 React TypeScript 開發的現代化國際象棋遊戲，具有 AI 對手、提示功能和美觀的淺色系界面。

## 功能特色

- ✅ **完整的象棋規則**：支持所有棋子的標準移動規則
- ✅ **AI 對手**：智能 AI 對手，具有評分系統和策略思考
- ✅ **雙人模式**：支持兩個人輪流下棋
- ✅ **遊戲提示**：為玩家提供移動建議
- ✅ **重新開始**：隨時重新開始遊戲
- ✅ **新遊戲**：快速開始新的一局
- ✅ **將軍檢測**：自動檢測將軍、將死和和棋
- ✅ **被吃棋子顯示**：顯示雙方被吃的棋子
- ✅ **響應式設計**：適配各種屏幕尺寸
- ✅ **現代化 UI**：淺色系設計，簡潔美觀

## 技術棧

- **React 18** - 前端框架
- **TypeScript** - 類型安全
- **Vite** - 構建工具
- **CSS3** - 樣式設計

## 安裝和運行

### 系統需求

- **Node.js**: 版本 16.0 或更高
- **npm**: 版本 8.0 或更高
- **現代瀏覽器**: Chrome, Firefox, Safari, Edge

### 1. 安裝依賴

首先確保您已經安裝了 Node.js 和 npm。然後在專案目錄中執行：

```bash
npm install
```

這將會安裝以下主要依賴套件：
- `react` (^18.2.0) - React 框架
- `react-dom` (^18.2.0) - React DOM 渲染
- `typescript` (^5.0.0) - TypeScript 編譯器
- `vite` (^4.5.14) - 快速構建工具
- `@vitejs/plugin-react` (^4.0.0) - Vite React 插件

### 2. 啟動開發服務器

```bash
npm run dev
```

啟動後，您會看到類似以下的輸出：
```
  VITE v4.5.14  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h to show help
```

在瀏覽器中打開 `http://localhost:5173/` 即可開始遊戲。

### 3. 構建生產版本

```bash
npm run build
```

這會創建一個 `dist` 資料夾，包含優化後的靜態檔案。

### 4. 預覽生產版本

```bash
npm run preview
```

這會啟動一個本地服務器來預覽構建後的應用程式。

## 遊戲規則

### 國際象棋基本規則

國際象棋是一個兩人對戰的策略遊戲，目標是將死對方的國王。

#### 棋盤設置
- 棋盤由 8x8 共 64 個方格組成
- 白方在下方，黑方在上方
- 每方各有 16 個棋子

#### 棋子移動規則

**國王 (King) ♔♚**
- 可以朝任何方向移動一格
- 不能移動到會被攻擊的位置
- 被攻擊時稱為「將軍」
- 無法逃脫將軍時稱為「將死」

**皇后 (Queen) ♕♛**
- 可以朝任何方向移動任意格數
- 結合了車和象的移動方式
- 是棋盤上最強的棋子

**車 (Rook) ♖♜**
- 可以水平或垂直移動任意格數
- 不能斜向移動
- 可以進行「王車易位」

**象 (Bishop) ♗♝**
- 只能斜向移動任意格數
- 白象只能在白格，黑象只能在黑格
- 不能跳過其他棋子

**馬 (Knight) ♘♞**
- 移動方式為「日」字形（2+1 或 1+2）
- 是唯一可以跳過其他棋子的棋子
- 移動時不受路徑阻擋

**兵 (Pawn) ♙♟**
- 向前移動一格，不能後退
- 第一次移動可以選擇移動一格或兩格
- 吃子時只能斜向吃子
- 到達對方底線可以升變為其他棋子

#### 特殊規則

**王車易位 (Castling)**
- 國王和車之間沒有其他棋子
- 國王和車都沒有移動過
- 國王沒有被將軍
- 國王移動兩格，車移動到國王旁邊

**兵的升變 (Promotion)**
- 兵到達對方底線時必須升變
- 可以選擇升變為皇后、車、象或馬
- 通常選擇升變為皇后

**吃過路兵 (En Passant)**
- 當對方兵第一次移動兩格時
- 如果經過我方兵的攻擊範圍
- 我方兵可以斜向吃掉對方兵

### 遊戲操作

#### 基本操作
1. **選擇棋子**：點擊要移動的棋子
2. **移動棋子**：點擊目標位置
3. **取消選擇**：再次點擊已選中的棋子

#### 遊戲模式
- **AI 模式**：與 AI 對手對戰（默認）
- **雙人模式**：兩個人輪流下棋

#### 控制按鈕
- **新遊戲**：開始一局全新的遊戲
- **重新開始**：重置當前遊戲
- **提示**：獲得移動建議
- **AI 模式/雙人模式**：切換遊戲模式
- **遊戲規則**：查看詳細的規則說明

#### 遊戲狀態
- **輪到白方/黑方**：顯示當前輪到哪一方下棋
- **將軍**：當國王被攻擊時顯示
- **將死**：當無法解除將軍時顯示
- **和棋**：當無法移動且未被將軍時顯示

## AI 對手說明

AI 對手使用簡單的評分系統：

- **吃子優先**：優先選擇能吃子的移動
- **中心控制**：偏好控制棋盤中心
- **安全考慮**：避免將自己的棋子置於危險位置
- **將軍獎勵**：給予將軍額外分數
- **隨機性**：添加隨機因素避免過於可預測

## 開發說明

### 添加新功能

1. 在 `types/chess.ts` 中定義相關類型
2. 在 `utils/chessLogic.ts` 中實現邏輯
3. 在 `hooks/useChessGame.ts` 中管理狀態
4. 在組件中使用新功能

### 自定義樣式

主要樣式在 `src/App.css` 中，使用 CSS 變量可以輕鬆調整顏色主題。
