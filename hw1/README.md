# 羅雲生 - 個人網站

這是一個專業的個人網站首頁，專為求職導向設計，展示個人技能、學經歷和聯絡資訊。

## 功能特色

- **響應式設計**: 支援桌面、平板和手機裝置
- **現代化UI**: 簡潔美觀的設計風格
- **互動動畫**: 滾動動畫、打字效果、技能條動畫
- **聯絡表單**: 完整的表單驗證和提交功能
- **導航功能**: 平滑滾動和移動端漢堡選單

## 檔案結構

```
hw1/
├── index.html          # 主要HTML結構
├── styles.css          # CSS樣式文件
├── script.js           # JavaScript功能
├── script.ts           # TypeScript源碼（可選）
└── README.md           # 說明文件
```

## 使用方式

1. 直接在瀏覽器中開啟 `index.html` 檔案
2. 所有功能都會自動載入和初始化

## 自訂內容

### 個人資訊
在 `index.html` 中修改以下內容：
- 姓名：搜尋 "羅雲生" 並替換
- 聯絡電話：搜尋 "+886-XXX-XXX-XXX" 並替換
- Email：搜尋 "your.email@example.com" 並替換
- GitHub：搜尋 "github.com/yourusername" 並替換
- LinkedIn：搜尋 "linkedin.com/in/yourprofile" 並替換

### 學經歷
在學經歷區塊中修改：
- 學校名稱和就讀期間
- 工作經驗和時間
- 專案經驗描述

### 技能
在技能區塊中調整：
- 程式語言和熟練度
- 技術框架
- 軟技能

## 技術規格

- **HTML5**: 語義化標記
- **CSS3**: Flexbox、Grid、動畫效果
- **JavaScript ES6+**: 類別、箭頭函數、模組化
- **響應式設計**: 移動優先設計原則
- **無外部依賴**: 除了字體和圖標庫

## 瀏覽器支援

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+
