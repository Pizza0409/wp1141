# NTU 課程選課系統

一個基於 React + TypeScript 的純前端課程選課系統，支援課程瀏覽、選課管理和送出記錄功能。

## 功能特色

### 🎓 課程瀏覽
- **關鍵字搜尋**：支援課程名稱、代碼、教師搜尋
- **條件篩選**：系所、學分、星期、時段、教師篩選
- **課程分組**：全部課程、通識課程、新生課程、學程課程
- **即時搜尋**：防抖動搜尋，優化使用者體驗
- **虛擬化列表**：高效渲染大量課程資料
- **衝堂檢查**：即時顯示課程衝突警告

### 📋 選課管理
- **預選課程**：加入預選清單，可隨時移除
- **衝堂檢查**：預選課程與已選課程衝突檢查
- **統計資訊**：預選課程數量、學分統計
- **送出選課**：確認選課並生成送出記錄
- **衝堂限制**：有衝堂時無法送出選課

### 📊 課程時間表
- **視覺化課表**：週一到週日的時間表顯示
- **多課程支援**：同時顯示預選和已選課程
- **衝堂警告**：時間衝突課程以紅色標示
- **無時間課程**：顯示沒有時間資訊的課程
- **星期顏色**：不同星期使用不同顏色區分

### 📝 送出記錄
- **選課歷史**：查看所有送出記錄
- **狀態管理**：已送出、已確認狀態
- **時間表顯示**：每個記錄的時間表預覽
- **記錄詳情**：課程列表、送出時間、備註

## 技術架構

### 前端技術
- **React 18** + **TypeScript**：現代化前端框架
- **Material-UI (MUI)**：UI 組件庫
- **Vite**：快速建置工具
- **PapaParse**：CSV 資料解析

### 專案結構
```
src/
├── components/          # React 組件
│   ├── CourseBrowser.tsx      # 課程瀏覽
│   ├── CourseCard.tsx         # 課程卡片
│   ├── CourseSelection.tsx    # 選課管理
│   ├── CourseSchedule.tsx     # 課程時間表
│   ├── SubmissionHistory.tsx  # 送出記錄
│   ├── VirtualizedCourseList.tsx # 虛擬化列表
│   └── ScrollToTop.tsx        # 回到頂部
├── context/             # React Context
│   └── CourseContext.tsx      # 課程資料狀態管理
├── hooks/               # 自定義 Hooks
│   ├── useCourseSearch.ts     # 課程搜尋邏輯
│   ├── useCourseSelection.ts  # 選課管理邏輯
│   └── useCSVReload.ts        # CSV 重新載入
├── types/               # TypeScript 類型定義
│   └── course.ts             # 課程相關類型
├── utils/               # 工具函數
│   └── csvParser.ts          # CSV 解析工具
└── App.tsx             # 主應用組件
```

### 資料流程
1. **CSV 載入**：從 `public/data/hw3-ntucourse-data-1002.csv` 載入課程資料
2. **資料解析**：使用 PapaParse 解析 CSV，轉換為 TypeScript 類型
3. **狀態管理**：使用 React Context 管理全域狀態
4. **組件渲染**：各組件透過 Context 和 Hooks 存取資料

## 使用說明

### 課程瀏覽
1. 在搜尋欄輸入關鍵字或使用篩選條件
2. 選擇課程分組（全部、通識、新生、學程）
3. 點擊課程卡片的「+」按鈕加入預選
4. 已選課程會顯示「已選」標籤且無法重複選擇

### 選課管理
1. 查看預選課程列表和統計資訊
2. 檢查課程時間衝突警告
3. 移除不需要的預選課程
4. 解決衝堂問題後送出選課

### 課程時間表
1. 查看預選和已選課程的時間安排
2. 識別時間衝突的課程
3. 查看無時間課程列表

### 送出記錄
1. 查看所有選課送出記錄
2. 查看每個記錄的詳細資訊
3. 預覽記錄的時間表

## 資料格式

### CSV 欄位說明
- `cou_cname`：課程中文名稱
- `cou_code`：課程代碼
- `tea_cname`：教師姓名
- `credit`：學分數
- `dpt_abbr`：開課系所
- `day1-day6`：上課時間（星期幾 + 節次）
- `clsrom_1-clsrom_6`：教室資訊
- `mark`：通識課程標記

### 時間格式
- `day1="12"`：星期一第1、2節
- `day2="567"`：星期二第5、6、7節
- `day3="AB"`：星期三第10、11節（A=10, B=11）
